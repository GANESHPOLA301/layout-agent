# Layout Agent — Compra AI Engineer Intern Assignment

A chat-based AI layout editor that transforms design JSON using natural language prompts.

## Features

- Chat-based layout editing
- JSON transformation engine
- Wireframe preview
- Undo / Redo support
- Suggested prompts
- JSON viewer with copy support
- Claude Sonnet integration

## Tech Stack

- React + Vite
- Claude Sonnet API
- Vanilla CSS

```bash
git clone https://github.com/YOUR_USERNAME/layout-agent.git
cd layout-agent
